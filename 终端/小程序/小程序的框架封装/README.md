# weChatProject
简单封装了一个微信小程序的框架

# 目录结构说明


├─assets  /*储存静态文件*/     
│  └─images  /*静态图片储存*/                
│      └─tab /*小程序tab图标切换*/             
├─components /*存储组件*/                    
├─config /*配置常用文件。比如二次封装小程序的接口以及路由，照片路径,api统一存储 */                
├─pages /*小程序页面编写 */                   
│  ├─Auth /*注册*/                    
│  ├─home /*首页*/             
│  │  └─Index                
│  ├─mine /*我的*/              
│  │  └─Index               
│  └─shop /*商城*/                 
│      └─Index             
└─utils /*放置插件以及封装的方法*/                        
    └─regenerator-runtime
